# Lost-and-Found-Web-Application-

A full-stack Lost and Found web application designed to help users report, search, and manage lost or found items efficiently.

## Features

- Report lost items
- Report found items
- Browse available listings
- Search items
- User authentication
- Secure access to protected routes
- Responsive user interface

## Tech Stack

Frontend
- React.js
- Material UI
- CSS

Backend
- Node.js
- Express.js

Database
- PostgreSQL

Authentication
- JWT (JSON Web Token)

Deployment
- Vercel

## Project Structure

```
client/
server/
database/
```

## Installation

```bash
npm install
npm start
```
